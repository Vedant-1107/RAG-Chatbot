# 🤖 RAG Chatbot - PDF Question Answering System

A production-ready Retrieval-Augmented Generation (RAG) chatbot that allows users to upload PDF documents and ask questions about their content.

## ✨ Features

- 📄 Upload multiple PDF documents
- 💬 Natural language question answering
- 🔍 Smart context retrieval with source citations
- 🧠 Conversation memory for follow-up questions
- ⚡ Real-time responses with loading indicators
- 🎨 Modern, responsive UI with TailwindCSS
- 🔒 No hallucinations - answers only from uploaded content

## 🛠️ Tech Stack

### Backend
- Python 3.9+
- FastAPI
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Groq API (LLaMA 3.1)

### Frontend
- React 18
- Vite
- TailwindCSS
- Axios
- Lucide Icons

## 📋 Prerequisites

- Python 3.9 or higher
- Node.js 16+ and npm
- Groq API key ([Get it here](https://console.groq.com))
